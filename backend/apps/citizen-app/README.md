## Temporary E2E Test App

This folder currently contains a minimal Expo/React Native test app for backend flow verification.

Run:

```bash
npm install
npm run web
```

or:

```bash
npm run ios
```

Default API URL:

```txt
http://localhost:8080
```

When testing on a physical phone, replace it in the app with your Mac's LAN address, for example:

```txt
http://192.168.0.10:8080
```

# MOFAapp Admin Web

현재 로컬 백엔드의 최신 단일 상담을 읽기 전용으로 확인하는 테스트 웹입니다. DB, 다중 채팅방, WebSocket 없이 `GET /chat/messages`를 1초마다 조회합니다.

## 실행

첫 번째 터미널에서 백엔드를 실행합니다.

```bash
cd backend
npm run dev
```

두 번째 터미널에서 관리자 웹을 실행합니다.

```bash
cd admin-web
npm run dev
```

브라우저에서 `http://127.0.0.1:4173`을 엽니다. 모바일 앱에서 AI 상담 메시지를 보내면 왼쪽에 현재 상담이 나타납니다. 상담을 선택하면 민원인과 AI 상담사의 전체 대화가 표시됩니다.

서버를 재시작하면 공유 중인 대화는 초기화됩니다. 이 웹은 한 번에 하나의 로컬 상담만 표시하는 프론트엔드 테스트용입니다.
